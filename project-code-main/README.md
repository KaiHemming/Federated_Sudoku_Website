# Group 5 Fantastic Puzzles Fife
Group 5's website, that's a part of the Aces Supergroup Federation, allowing cross-federation features such as:
- Puzzle upload and download
- cross-federation logins

Support puzzles
- Sudoku
- Miracle Sudoku
- MineSweeper
